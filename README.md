# Worth-It

Worth-It is an AI spend optimization platform that helps startups and teams audit their AI tooling stack, identify overspending opportunities, and generate intelligent optimization recommendations.

## Features

- Dynamic AI spend audit workflow
- Intelligent recommendation engine
- Savings estimation system
- Executive AI summaries
- Public shareable audit reports
- Supabase cloud persistence
- Responsive modern SaaS UI
- Real-time optimization insights

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Supabase
- React Hook Form
- UUID
- Vercel

## How It Works

1. Users input their AI tooling stack
2. Worth-It analyzes plans, pricing, and usage patterns
3. The audit engine identifies optimization opportunities
4. Reports are generated dynamically and stored in Supabase
5. Public report URLs can be shared and revisited
