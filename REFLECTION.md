# Reflection

Building Worth-It helped me understand how modern SaaS products combine frontend UX, backend persistence, business logic, and deployment workflows.

One of the biggest learnings was designing a recommendation engine that balances realistic optimization suggestions with a clean user experience.

I also learned how important product polish is. Features like loading states, public report sharing, and executive summaries significantly improved the perceived quality of the application.

If I continued building this project, I would explore:
- Real LLM-powered summaries
- Team collaboration features
- Historical audit tracking
- Stripe billing integration
- Real-time pricing APIs
